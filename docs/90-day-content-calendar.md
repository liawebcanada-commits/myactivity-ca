## Phase 1 — Weeks 1–2: Let Indexation Stabilize

**No new content.** Google needs time to crawl and assess the initial 312 pages before you introduce new signals that could confuse the crawl budget or create indexation noise.

**Monitoring tasks (weekly):**
- GSC: Check Coverage report for errors (specifically hreflang mismatches, canonical conflicts, Soft 404s)
- GSC: Verify both locale sitemaps (`/sitemap/en.xml` and `/sitemap/fr.xml`) are submitted and processing without errors
- GA4: Confirm real-time data is flowing; verify `affiliate_click` events are firing on test click-throughs
- PageSpeed Insights: Run city hub template (highest-traffic page type) on mobile — confirm LCP < 2.5s, CLS < 0.1
- Check `robots.txt` at production URL is serving correctly
- Verify `canonical` and `hreflang` in page `<head>` for 5–10 sampled pages using browser DevTools

**Success signals by end of Week 2:**
- All 312 pages submitted in sitemap (no 404s in GSC Coverage)
- Zero hreflang errors in GSC International Targeting report
- GA4 showing sessions from both /en/ and /fr/ paths

## Phase 2 — Weeks 3–6: Editorial Guides (Link-Building Bait)

Publish 4 long-form editorial guides (Section 9.3). These are the pages to pitch for backlinks — not category pages. Each guide should be 800–1,200 words minimum, address 3+ People Also Ask questions from the target SERP, and include genuine internal links to relevant city hub and category pages.

**Guide 1 — Week 3**
- **Title (EN):** The Ultimate Guide to Free Winter Activities Across Canada
- **Title (FR):** Guide ultime des activités hivernales gratuites au Canada
- **Slug:** `/en/guide/free-winter-activities-canada` + FR equivalent
- **Target keywords:** "free winter activities Canada", "activités gratuites hiver Canada"
- **Internal links:** Ottawa Rideau Canal, Quebec City Carnival, Winnipeg Forks Trail, Calgary Prince's Island Park
- **Link-building targets:** Pitch to Destination Canada, Today's Parent, Ottawa Citizen

**Guide 2 — Week 4**
- **Title (EN):** Best Kid-Friendly Activities in Canada's Major Cities
- **Title (FR):** Meilleures activités pour enfants dans les grandes villes canadiennes
- **Slug:** `/en/guide/kid-friendly-activities-canada`
- **Target keywords:** "kid friendly activities Canada", "activités enfants Canada"
- **Internal links:** Toronto ROM, Calgary Zoo, Vancouver Stanley Park Aquarium, Edmonton Valley Zoo
- **Link-building targets:** Pitch to Today's Parent, Maman Loup

**Guide 3 — Week 5**
- **Title (EN):** 10 Underrated Canadian Cities for a Weekend Getaway
- **Title (FR):** 10 villes canadiennes sous-estimées pour une escapade de fin de semaine
- **Slug:** `/en/guide/underrated-canadian-weekend-getaways`
- **Target keywords:** "weekend getaway Canada", "escapade fin de semaine Canada"
- **Internal links:** Hamilton waterfalls, Chambly canal, Halifax boardwalk, Kitchener-Waterloo Oktoberfest
- **Link-building targets:** Narcity Canada, BlogTO

**Guide 4 — Week 6**
- **Title (EN):** How to Spend a Perfect Day in Montreal (All Seasons)
- **Title (FR):** Comment passer une parfaite journée à Montréal (toutes saisons)
- **Slug:** `/en/guide/perfect-day-montreal`
- **Target keywords:** "things to do Montreal one day", "que faire une journée Montréal"
- **Internal links:** Montreal Old Port, Mont-Royal, Jean-Talon Market, Just For Laughs
- **Link-building targets:** Tourisme Montréal, La Presse Sorties

**Content process for each guide:**
1. Pull top 10 ranking pages for target query; extract People Also Ask questions
2. Outline: intro + 3 PAA answers + curated activity recommendations with internal links
3. Write at 800–1,200 words (human-quality; no AI boilerplate)
4. Add `FAQPage` JSON-LD only if Q&A content is substantial (Section 9.4)
5. Publish to both `/en/guide/[slug]` and `/fr/guide/[slug]`
6. Add guide to sitemap (auto-generated by `app/sitemap.ts` if guide data files are added)

## Phase 3 — Weeks 7–10: City Matrix Expansion

**Decision rule:** Expand based on which Phase 1 cities show the fastest indexation + impressions in GSC, not by guesswork.

**Week 7:** Pull GSC Impressions by city page (filter by URL path). Identify the 3–5 cities with the highest impressions-to-clicks ratio (high impressions + low clicks = strongest opportunity, likely ranking Page 2–3).

**Target expansion cities (candidates, ranked by population/search volume):**
1. Saskatoon, SK
2. Victoria, BC
3. London, ON
4. Kelowna, BC
5. Moncton, NB
6. Sherbrooke, QC (strong FR-CA long-tail opportunity)
7. Laval, QC (suburb adjacency to Montréal searches)
8. Mississauga, ON (suburban Toronto adjacency)

**Process to add a new city:**
1. Create `/data/activities/{city-slug}.json` with 5 activities (same schema)
2. Add city entry to `/data/cities.json`
3. No code changes required — `generateStaticParams()` in all page templates will auto-include the new city on next `next build`
4. Run `npm run validate-hreflang` and `npm run check-duplicate` before deploying

**Weeks 8–10:** Write activity data files for the chosen cities. Prioritize content quality over speed — each description must pass the 15% n-gram similarity check in CI.

## Phase 4 — Weeks 11–13: Health Report + Optimization

**Week 11:** Run the first monthly SEO health report.

```bash
npx ts-node --project tsconfig.json scripts/seo-health-report.ts
```

Review `docs/seo-health-report-{date}.md` and action any:
- Thin content (< 150 words) — rewrite flagged descriptions
- Stale event dates — update `event_start_date` in activity JSON files
- `#AFFILIATE_LINK` stubs — replace with live affiliate URLs once accounts are approved

**Week 12:** Underperformer analysis in GSC.
- Filter: Impressions > 50, Position > 20 (ranking but not converting clicks)
- For each underperforming city+category page: add a more specific H2, expand the intro copy, improve internal links
- Priority: city hub pages where Position is 11–20 (just outside page 1)

**Week 13:** Review and iterate.
- Is `/fr/` receiving traffic proportional to expected bilingual audience (~20–30% of Canadian internet traffic is French)? If under, consider adding more FR-specific editorial content.
- Check `affiliate_click` event data in GA4 — which networks/cities/categories have the highest click-through rate? Prioritize those cities for expansion in Phase 3 iteration.
- Decide whether to expand to Tier 2 cities (Phase 3 candidates) based on performance data.
